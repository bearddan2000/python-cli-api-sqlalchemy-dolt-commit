# python-cli-api-sqlalchemy-dolt-commit

## Description
A POC for dolt+sqlalchemy.

## Tech stack
- python
- sqlalchemy

## Docker stack
- python:latest
- dolthub/dolt-sql-server

## To run
`sudo ./install.sh -u`

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
